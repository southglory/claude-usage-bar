Then build the font

uv run --with fonttools \
  python tools/build_mascot_font.py \
  mascot.json mascot.ttf mascot
That prints the contributes.icons block and the characterFrames value to paste into the extension. Drop mascot.ttf next to package.json, repackage, and your mascot animates in the status bar.

No rebuild? Pixel mascots need a bundled font (above). For a quick change without rebuilding, set claudeMultiUsage.characterFrames to emojis or codicons, e.g. ["▃","▆"] or ["$(rocket)","$(flame)"].
