Then build the font

uv run --with fonttools \
  python tools/build_mascot_font.py \
  mascot.json mascot.ttf mascot
Prints the contributes.icons block and the characterFrames value to paste in. Drop mascot.ttf next to package.json and repackage.

Colors are for your design only. The status-bar mascot is a single-color font glyph, so every painted pixel becomes a solid silhouette (no holes) — colors help you draw, but the glyph takes its color from the VS Code theme.
